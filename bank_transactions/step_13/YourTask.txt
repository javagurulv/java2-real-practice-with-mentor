Наш вариант интерфейса для нахождения текущего баланса вы найдёте:

org/javaguru/banktransactions/CurrentBalanceCalculator.java

Данный интерфейс подчёркивает своё назначение - подсчёт
текущего баланса (в названии).

В интерфейсе объявлен один метод

int calculate(BankAccount bankAccount);

который как входной параметр получает объект BankAccount
и выдаёт в качестве результата int (баланс в центах).

Данный интерфейс подчёркивает выдвинутое к нему требование - расчёт
текущего баланса банковского аккаунта.


Ваша следующая задача: воспользовавшись созданным интерфейсом
написать автоматические тесты для расчёта текущего баланса
банковского аккаунта.

Автоматические тесты являются признаком профессионального решения.

Хорошие автоматические тесты являются признаком мастерства!
