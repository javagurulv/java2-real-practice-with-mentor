Задача: Подсчет суммы всех простых чисел в заданном диапазоне.

Описание задачи:
Необходимо написать многопоточное приложение
на Операционной Системе с Многозадачностью,
которое будет эффективно находить и складывать
все простые числа в заданном диапазоне.

Простые числа - это натуральные числа больше 1,
которые делятся только на себя и на 1.

Инструкции:
1. Пользователь должен ввести начальное
   и конечное значение диапазона для поиска простых чисел.
2. Приложение должно создать несколько потоков (по умолчанию,
   например, 4 или равное числу ядер CPU) для параллельной обработки диапазона.
3. Каждый поток должен проверять числа в своем поддиапазоне и
   добавлять найденные простые числа к общей сумме.
4. В конце выполнения, программа должна вывести сумму всех
   найденных простых чисел.

Эта задача идеально подходит для многопоточного приложения,
так как поиск простых чисел в больших диапазонах может быть
ресурсо емкой операцией, и параллельная обработка может
значительно ускорить выполнение задачи.

Пример решения этой задачи может включать в себя создание
классов потоков для обработки диапазонов чисел,
использование механизмов синхронизации для обновления общей суммы,
и обработку ввода пользователем для определения диапазона чисел.
