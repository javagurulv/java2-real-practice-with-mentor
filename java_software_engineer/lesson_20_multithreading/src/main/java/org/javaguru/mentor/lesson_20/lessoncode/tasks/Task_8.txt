Задача: Многопоточная загрузка и обработка текста из интернета.
(Site crawling)

Описание задачи:
Разработайте многопоточное приложение, которое загружает
текст из интернета и обрабатывает его параллельно.

Инструкции:
1. Пользователь вводит список URL-адресов сайтов, с которых он хочет загрузить текст.
2. Приложение создает несколько потоков (по количеству URL-адресов) для параллельной загрузки.
3. Каждый поток загружает текст с заданного URL-адреса и сохраняет его на диск или в память.
4. После загрузки каждый текст обрабатывается (например, например составляя список уникальных слов).
5. Завершив обработку всех сайтов, приложение может объединить результаты в один файл или вывести на консоль.

Желательно реализовать функцию загрузки страниц всего указанного сайта.
Например, если на загружаемой странице встретилась ссылка на другую
страницу этого же сайта, то и ее нужно тоже загрузить и обработать.

При реализации этой функциональности у вас получится полноценный
web crawler, который может постранично загрузить контент указанного
сайта начиная с главной его страницы.

Эта задача подразумевает использование многопоточности
для эффективной загрузки и обработки множества страниц параллельно,
что может сэкономить время и ресурсы при выполнении подобных задач.
