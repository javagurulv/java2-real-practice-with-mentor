Ваша задача: задача 7: создание acceptance тестов.

Подумайте и составьте тест план.
Тест план должен включать в себя описание конкретных
тестовых сценариев, которые вы хотите протестировать.
После создания тест плана, создайте автоматические
тесты для каждого тестового сценария из тест плана.

PS: в процессе работы над созданием тест плана подумайте
о целесообразности разделения всех тестов на блоки:
- блок тестов для общих полей уровня полиса;
- блок тестов специфических для TRAVEL_MEDICAL риска;
- блок тестов специфических для TRAVEL_CANCELLATION риска;
- блок тестов общих для TRAVEL_MEDICAL и TRAVEL_CANCELLATION рисков.
Вполне возможно так будет проще ориентироваться в структуре тестов.
