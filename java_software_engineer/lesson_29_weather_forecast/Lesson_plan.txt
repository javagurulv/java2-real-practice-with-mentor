Проект “Прогноз погоды”

Веб-приложение, показывающее прогноз погоды.

Идея: пользователь открывает веб страницу приложения.
На странице в форме ввода вводит название города или страны
и нажимает кнопку "Показать погоду".

Приложение используя https://openweathermap.org/ и его API
узнаёт погоду в указанном пользователем городе или стране
и показывает ее на веб странице.

Цель проекта:
Познакомиться с синхронным способом интеграции приложений
используя HTTP/REST.

Полезные ссылки:
https://openweathermap.org/

https://openweathermap.org/api

https://openweathermap.org/api/geocoding-api - API для получения
координат по названию города или страны.

https://openweathermap.org/api/one-call-3 - API по получению
прогноза погоды по координатам.

Для использования https://openweathermap.org нужно зарегистрироваться
на сайте и получить API ключ. Ключ даёт право на 1000 бесплатных
запросов в сутки.
