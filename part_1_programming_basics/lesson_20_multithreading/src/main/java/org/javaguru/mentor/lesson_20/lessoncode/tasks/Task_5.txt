Задача: Параллельный поиск наибольшего числа в большом массиве.

Описание задачи:
Напишите многопоточное приложение для поиска наибольшего числа
в массиве данных. Массив содержит миллионы элементов,
и цель состоит в том, чтобы максимально эффективно найти
наибольшее число, разбивая массив на под массивы и обрабатывая
их параллельно.

Инструкции:
1. Создайте большой массив случайных чисел.
2. Определите количество потоков, которые будут
   использоваться для обработки массива.
3. Разделите массив на под массивы, присвоив каждому
   потоку свой участок данных для анализа.
4. Каждый поток должен найти наибольшее число в своем под массиве.
5. После завершения выполнения всех потоков, найдите наибольшее
   число среди результатов, найденных каждым потоком.

Эта задача хорошо подходит для многопоточной обработки,
так как обработка больших объемов данных может быть медленной
на одно поточных приложениях.

Параллельная обработка позволяет эффективно использовать
ресурсы процессора и ускорить выполнение задачи.

Пример решения может включать в себя создание классов потоков
для обработки под массивов данных, использование средств
синхронизации для сбора результатов и выбора наибольшего числа.
