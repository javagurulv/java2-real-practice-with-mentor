Multithreading theory topics:

1. Overall computer system hardware architecture
   (CPU, RAM, HDD/SSD, IO devices, mother board).
   How they related to each other and how work together?

2. One core CPU (CPU internals, ALU, скорость работы/операций в секунду,
   транзисторы и их размер (нанометры), технологии создания CPU ASML).

3. Multi core CPU (2/4/6/8/...32/64/...).

4. What is a resource? (CPU core time, RAM, HDD, network, etc).

5. OS (Operating System), Process concept. how OS manage resources
   (RAM, CPU time, file system, network, etc)?

6. How application run the code? Thread concept.
   (OS kernel level vs user level threads).

7. OS thread scheduling
   (https://www.javatpoint.com/threads-in-operating-system, https://www.javatpoint.com/os-scheduling-algorithms)

8. Java Runnable interface.

9. Java Thread class (creation, launching, execution, finishing).

10. JVM memory: Heap.

11. JVM memory: Stack.

12. Shared resources (общие ресурсы и проблемы
    их одновременного использования).

13. Synchronized.


Дополнительные темы для изучения:
- варианты синхронизации;
- atomic классы;
- пулы потоков.
